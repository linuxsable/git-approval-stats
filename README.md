# git-approval-stats

Run a script to see who has reviewed the most code in your repo.

NOTE: This only works in BitBucket cloud currently.

## Usage

```
node git-approval-stats.js <REPO DIR>
```

```

  2. Bob Decker (1063 approvals)

  ...
```
